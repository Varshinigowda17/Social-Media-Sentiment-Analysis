Project Overview
-This project is an end-to-end Data Analytics and Machine Learning project that analyzes social media posts to understand public opinion and customer sentiment. By processing textual data, the project identifies whether a post expresses positive, negative, or neutral sentiment.
-The project demonstrates the complete data analytics workflow—from data collection and cleaning to exploratory analysis, machine learning model development, and interactive dashboard creation. The goal is to transform raw social media data into meaningful insights that support better business decisions

Tools Used
-Python
-Pandas
-NumPy
-Matplotlib
-Scikit-learn
-SQL
-Power BI
-Google Colab

Workflow
-Phase 1: Data Collection
  Loaded the social media sentiment dataset.
-Phase 2: Data Cleaning
  Removed missing values.
  Cleaned and prepared the dataset for analysis.
-Phase 3: Exploratory Data Analysis (EDA)
  Analyzed sentiment distribution.
  Explored platform-wise and category-wise trends.
  Created visualizations for better understanding.
-Phase 4: Machine Learning
  Converted text into numerical features.
  Trained a sentiment classification model.
  Evaluated the model using performance metrics.
-Phase 5: Dashboard
  Developed an interactive Power BI dashboard to present the analysis and insights

Results
 -Successfully cleaned and prepared the dataset.
 -Identified positive, negative, and neutral sentiment patterns.
 -Built a machine learning model for sentiment classification.
 -Created an interactive Power BI dashboard for visual analysis.
 -Generated insights that help understand customer opinions and trends on social media.
